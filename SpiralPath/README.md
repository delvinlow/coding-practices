## Problem
Given a positive integer `n`, generate an `n x n` matrix filled with elements from 1 to n ^ 2 in spiral order.

---

### Example 1
**Input:**

`n = 3`

**Output:**

```python
[[1, 2, 3], 
 [8, 9, 4], 
 [7, 6, 5]]
```

### Example 2
**Input:**

`n = 5`

**Output:**

```python
[[1, 2, 3, 4], 
[12, 13, 14, 5], 
[11, 16, 15, 6], 
[10, 9, 8, 7]]
```
